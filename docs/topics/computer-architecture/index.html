Dalla Fisica al Calcolo Binario

Guida interattiva per comprendere l'evoluzione dell'hardware digitale.

1. Il Transistor MOSFET (Interruttore Fisico)

Il transistor ha 3 pin: Gate (Comando), Source (Ingresso corrente) e Drain (Uscita corrente). La tensione applicata al Gate crea un campo elettrico che "attira" cariche negative creando un canale di silicio conduttivo.

Stato: Isolante. Nessuna corrente passa.

2. Le Porte Logiche Usate nei Circuiti

I meccanismi interni delle porte usate nei circuiti successivi:

Porta AND

Meccanismo: Due transistor in serie. La corrente passa solo se entrambi i transistor hanno il Gate attivo ($1$).

Regola: $1 \text{ AND } 1 = 1$, altrimenti $0$.

Porta XOR (OR Esclusivo)

Meccanismo: Confronta due ingressi. Restituisce $1$ solo se gli ingressi sono diversi tra loro.

Regola: $1 \text{ XOR } 0 = 1$, $1 \text{ XOR } 1 = 0$, $0 \text{ XOR } 0 = 0$.

Porta NOT (Inverter)

Meccanismo: Un transistor che quando si attiva scarica il segnale a terra ($0$). Quando è spento, l'uscita rimane alta ($1$).

Regola: Inverte il bit ($0 \rightarrow 1$, $1 \rightarrow 0$).

3. Circuiti Logici Complessi

A) Half Adder (Sommatore Binario)

Calcola la somma ($A + B$). La porta XOR produce il bit di Somma, la porta AND produce il Riporto (Carry).

Somma (XOR): 0 | Riporto (AND): 0

B) Comparatore "Maggiore" ($A > B$)

Determina se il bit $A$ è maggiore del bit $B$. Poiché lavoriamo a 1 bit, $A > B$ è vero solo se $A=1$ e $B=0$. Si usa una porta AND unita a una porta NOT ($A \text{ AND (NOT } B)$).

Risultato ($A > B$): FALSO (0)

C) Multiplexer (Selettore 2-a-1)

Sceglie quale tra due dati ($D0$ o $D1$) inviare in uscita, in base al segnale di Selezione (S).

Uscita Finale: 0

D) Flip-Flop SR (1 Bit di Memoria)

Usa il Retroeffetto (Feedback): le uscite delle porte sono collegate agli ingressi opposti per "incastrare" il segnale e ricordare lo stato anche dopo che il comando cessa.

Stato Memorizzato (Q): 0
